Descripcion: 
El proyecto adjunto consta de una app de presentacion de un proyecto musical, en terminos abstractos podria ser simplemente la presentacion de una "marca" (brand), se ha tomado de referencia la pagina de https://www.wosds3.com tanto como se ha dejado apertura a la imaginación al momento de preparar el sketch.
En termimnos de los temas aplicados se han utilizado las tecnologias de handlebars y bootstrap combinadas a un css local para el desarrollo del frontend y se han sacado provecho de las posibilidades de bootsrap en tanto scrips de JS para el deploy veloz de carruseles y formularios. Asimismo se aplicaron los conocimientos adquiridos de MYSQL para lograr un formulario que en terminos locales ha funcionado perfectamente.
En terminos de backend se ha dezplegado todo a partir de nodejs por la utilidad en tanto optimizacion de codigo.
